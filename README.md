# Ducked

App launcher that doesn't aim for being Ultra Shitty. Python, GTK, bash, sqlite, kittens.

Supports basic search for:

    Google
    Youtube
    Wiki
    4chan
    Github
    Google Maps
    Gmail
    torrentz
    Twitter
